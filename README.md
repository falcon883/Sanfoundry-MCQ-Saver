# Sanfoundry-MCQ-Saver
### Saves all the mcqs to a pdf file.
#### Keep sanUrls.py and sanfoundry.py in the same folder.

## Usage:

```
py sanfoundry.py
```
Script uses below types of link.

For downloading single mcq page.
